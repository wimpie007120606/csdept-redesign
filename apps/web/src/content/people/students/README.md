# Postgraduate students data

- **Doctoral students:** Paste your JSON array into `doctoral.json`.
- **Masters students:** Paste your JSON array into `masters.json`.

Each item must have:

- `name` (string)
- `supervisor` (string)
- `topic` (string)

Slugs are generated automatically from `name` (e.g. "Jane Doe" → `jane-doe`).
