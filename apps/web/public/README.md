# Web public assets

Static files here are served at the site root. Keep assets organized so the site keeps working.

## Folders

| Folder | Purpose | URL path |
|--------|---------|----------|
| **people/** | Staff profile photos (People page, profiles, research groups) | `/people/...` |
| **research-groups/** | Research group wallpapers/banners | `/research-groups/...` |
| **brand/** | Logos and brand assets (e.g. Stellenbosch logo) | `/brand/...` |
| **images/** | Other images (see `images/README.md`) | `/images/...` |
| **bridging/** | Bridging course assets | `/bridging/...` |

## Root-level files

- `realbackground2.jpg`, `realbackground3.jpeg`, `background.jpg` — hero/background images
- `newlogo.jpeg` — footer logo
- `first-year-science-student-asking-question-chemistry.jpg` — undergraduate page
- **`science-yearbook.pdf`** — Faculty of Science Yearbook (Undergraduate page “Science Yearbook” card). Add this file here for View/Download to work.

Staff photos use naming like `FirstName_LastName_People.jpg`; see `people/` and `images/README.md`.
